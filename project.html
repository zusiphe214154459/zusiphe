Our Project

We as team 1 members we are going to create an HTML website that will allow users (domestic workers and employers) to find jobs and employees respectively. This website will use a register or sign in and log in functions, technologies that we are going to use are Java and MySQL platforms. After the login the user (job seeker/employer) will be able to upload his/her CV or the job specifications respectively.
